## **Flight Booking System**

## **Overview:**
A console-based flight booking system that is designed to assist Air New Zealand in managing flight schedules and booking operations. The system includes administrative tasks while also improving overall customer experience by providing an efficient and user-friendly way to search, book, and manage flights.

## **Organisation:**
Air New Zealand 

## **Technologies Used:**
* C#
* .NET 8.0 or later
* Visual Studio Code

## **How to Run:**
1. Clone the repository GitHub URL: https://github.com/Ianieee36/ENSE706_Project
2. Open the project folder in Visual Studio or VS Code
3. Build and run the application

## **System Features:**
* User login for registered users, such as Customer and Admin.
* Registration for unregistered users, such as guests
* Guests can search flights and view flight details.
* Admins add, update, and remove flights, and retrieve all flights and track flight occupancy
* Customers can search and view flights, book a flight, and manage their bookings.
